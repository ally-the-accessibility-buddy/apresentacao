# Ally (The Accessibility Buddy)
## 1 - Delimitação Inicial do Domínio:
- Local: IFSC Campus SJ
- Clientes: Desenvolvedores Frontend, designers UI/UX, estudantes e curiosos.

- Descrição: Aplicativo Web de Identificação de erros de interfaces e acessibilidade, voltado para designers web e desenvolvedores web. O app terá desafios para os usuarios resolverem, os desafios serão dados como concluídos somente se o usuário corrigir corretamente a interface. Educando e incentivando boas práticas de desenvolvimento.
